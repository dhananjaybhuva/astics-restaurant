# Astics Node Task Project

## Prerequisite (Development)

| Module | Version |
| --- | --- |
| Node | 16.14.2 |
| Npm | 9.6.2 |
| pg | 8.7.1 |
| sequelize | 6.3.2 |
| Mongodb | 4.4 |

## Running Project In Local
``` bash
$ git clone <gitUrl>

$ Import postman collection /PostmanCollectionFile/Astics.postman_collection.json

$ npm start

```
